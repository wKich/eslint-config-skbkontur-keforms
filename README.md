## How to use

- Run `npm install eslint-config-skbkontur-keforms`
- Extend your local `.eslintrc.json` config:

  ```
  {
    "extends": [ "eslint-config-skbkontur-keforms" ]
  }
  ```
- (Optional) Setup plugins for your favorite code editor
